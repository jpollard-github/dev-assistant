# Dev Assistant CLI

Package contents for the `dev-assistant` command-line workflow.

Common commands after install:

```sh
dev-assistant doctor
dev-assistant init
dev-assistant run "fix the failing test"
dev-assistant review
```

Repository-level setup, local model runtime guidance, and workflow examples live in the root [README](../../README.md) plus:

- [`docs/local-model-runtimes.md`](../../docs/local-model-runtimes.md)
- [`docs/example-workflows.md`](../../docs/example-workflows.md)
